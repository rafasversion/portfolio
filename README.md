# eladev.com

Portfólio inspirado na estética windows retrô nos tons de rosa e azul.
Na versão desktop, o modo de visualização é windows e no mobile, estética de smartphone.

Tentei fazer ele simples, mas que tivesse ainda um pedacinho da minha personalidade.

## Stack

- React
- TypeScript
- Vite
